this file contains instructions on how to run the Thuisgalerie assignment from the University
of Applied Sciences Utrecht.

dependent plugins
================================
PIL 
pip install Pillow

requests
pip install requests

if needed
-------------
tkinter
pip install tkinter


running the application
===============================
To start this application you have to run the app.py file. After the application has started
you can choose between "Bezoeker" and "Galerie houder". And you will be send to their respective
pages and can do the different things they can do.
